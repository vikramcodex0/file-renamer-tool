File Renamer Tool 🚀

A simple Python CLI tool to rename multiple files in a folder efficiently.

---

🔧 Features

- Bulk file renaming
- Custom prefix support
- Auto numbering (1, 2, 3...)
- File type filter (.jpg, .png, etc)
- Clean and simple CLI usage

---

▶️ Usage

Run this command in terminal:

python main.py --folder test --prefix file --start 1 --ext .jpg
